# Multi-Class Stress Level Detection

## Overview
This project predicts stress levels (Low, Medium, High) using Heart Rate Variability (HRV) features.

## Features Used
- HR (Heart Rate)
- RMSSD
- LF/HF ratio
- SDRR
- Other physiological parameters

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn

## My Contribution
- Data preprocessing and feature selection
- Model building and training
- Evaluation of multi-class classification

## Key Learning
Learned how to build an end-to-end machine learning pipeline and handle multi-class classification problems.

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the model:
   python main.py
